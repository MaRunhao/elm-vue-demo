# 仿饿了么app点餐前端页面

> 马润豪

## 项目描述

 
  该项目使用Vuejs2.0进行搭建，webpack打包编译。UI等原型内容使用网络上饿了么设计图稿，页面响应式对不同分辨率手机进行了适配(media query)，后端接入数据使用 mock.js 进行模拟。
  引用第三方组件： [better-scroll](https://github.com/ustbhuangyi/better-scroll)
